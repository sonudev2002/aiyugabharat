# Aiyuga Bharat

Django website for `aiyugabharat.com`.

## Run Locally

```powershell
.\venv\Scripts\python.exe manage.py migrate
.\venv\Scripts\python.exe manage.py runserver
```

Open `http://127.0.0.1:8000/`.

## Project Structure

- `aiyugabharat/` - Django project settings and root URLs
- `home/` - homepage app, template, and static assets
- `home/static/home/images/hero-aiyuga-bharat.png` - generated hero visual
